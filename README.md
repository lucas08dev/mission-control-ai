# Mission Control AI

## Descrição
Sistema inteligente de monitoramento de missão espacial desenvolvido em Python.

## Objetivo
Monitorar ciclos de uma missão espacial e identificar riscos operacionais.

## Funcionalidades
- Monitoramento de temperatura
- Comunicação
- Bateria
- Oxigênio
- Estabilidade operacional
- Cálculo de risco
- Relatório final automático

## Tecnologias utilizadas
- Python
